## 三、接口测试结果

### 3.1 订单接口测试

```
POST   /api/orders          ✅ 创建订单成功，返回订单ID
GET    /api/orders          ✅ 返回当前用户订单列表
GET    /api/orders/:id      ✅ 返回单条订单详情
PUT    /api/orders/:id/status  ✅ 状态流转正常
PUT    /api/orders/:id/cancel  ✅ 取消成功并回滚库存
DELETE /api/orders/:id      ✅ 删除已完成/已取消订单成功
```

